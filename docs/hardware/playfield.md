# Физическое игровое поле

## Компоновка

Каждая сторона игрока:
- **7 ячеек для существ**
- **1 ячейка для героя**

Итого: 16 ячеек на поле (8 на каждого игрока).

## Аппаратура ячейки существа

- **RFID-считыватель** — идентифицирует карту в ячейке
- **RGB-светодиоды:** 12 слева + 12 справа + 8 сверху + 8 снизу = **40 светодиодов**

```
        [  8 сверху  ]
[12   ]  [  карта   ]  [12   ]
[лево ]  [   RFID   ]  [право]
        [  8 снизу  ]
```

> **Полосы:** левая `12` — здоровье, правая `12` — атака, верхняя `8` — ключевые слова/статусы. **Нижняя полоса (8 LED) зарезервирована** — в текущей версии не несёт функции, её назначение будет определено позже. Назначение полос и анимации — [[led-conventions|Конвенции LED]].

## Аппаратура ячейки героя

Ячейка героя использует **иную раскладку**, чем ячейка существа:

- **RFID-считыватель** — идентифицирует карту героя
- **RGB-светодиоды:** полоса `30` LED (здоровье) + полоса `10` LED (мана) = **40 светодиодов**

```
[  30 LED — здоровье героя  ]
        [   карта   ]
        [   RFID    ]
[      10 LED — мана       ]
```

> Иные индикаторы героя (сила героя, фракционные пассивки) в текущей версии не предусмотрены. **Оружие героя — опциональный, ещё не финализированный модуль**; при его принятии ячейку героя можно расширить дополнительными светодиодами под атаку/прочность оружия (аппаратного потолка `40` LED на ячейке героя нет).
