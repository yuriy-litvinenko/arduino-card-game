# Палитра тем сеттинга

Справочник тематических «кирпичей» для конструирования мира и фракций. Палитра делится на два сорта:

- **Тех-эстетики** — выбираются по **одной на фракцию** (определяют, как фракция выглядит и какой технологией «отвечает на конец света»).
- **Сквозные оси** — модификаторы, навешиваемые **поверх** любой тех-эстетики: идеология фракции, среда мира, тон.

Формула фракции: **тех-эстетика + (опц.) идеология + (опц.) тон**, в заданной **среде** мира.

> Не-фэнтези. Все темы — грунтованный sci-fi; «странное» объясняется аномалиями/физикой, а не магией.

## Тех-эстетики (1 на фракцию)

### Энергия / индустрия
- **АТОМПАНК** — ретро-ядерная эра, реакторы, счётчики Гейгера
- **ДИЗЕЛЬПАНК** — тяжёлая индустрия, копоть, броне-машины
- **РУСТПАНК / ИНДАСТРИАЛ-СКАВЕНДЖ** — ржавчина, лом, самоделки

### Жизнь / биология
- **БИОПАНК** — живые машины, выращенные органы, wetware
- **МУТАЦИИ / ГЕНПАНК** — направленная эволюция, химеры
- **СОЛЯРПАНК** — зелёные технологии, симбиоз, свет
- **МИКОПАНК** — мицелиальные сети, споровые «вычисления»
- **КЛИ-ФАЙ / КЛИМАТ-КОЛЛАПС** — засуха, потоп, экокатастрофа

### Цифра / разум
- **КИБЕРПАНК** — неон, импланты, корпорации, сети
- **ИИ / ПОСТ-СИНГУЛЯРНОСТЬ** — пробудившийся или умерший сверхразум
- **РОБОПАНК** — автономные машины, дроны
- **НАНОПАНК** — нанороботы, «серая слизь»
- **ТРАНСГУМАНИЗМ** — загрузка сознания, киборги, рои разумов
- **СЛЕЖКА / СИГНАЛ-ПАНК** — тотальный надзор, инфо-война

### Распад / выживание
- **ПОСТАПОКАЛИПСИС** — руины, выживание
- **РОЕВОЙ АГРО / БАНДЫ** — мародёры, числом

### Странное / физика
- **АНОМАЛИИ** — зоны, где ломаются правила
- **КВАНТОВЫЙ РАСПАД** — нестабильная реальность, эхо, расфазировка
- **КОСМИЧЕСКИЙ ХОРРОР** — чужое и непостижимое *(по желанию)*

### Космос / мегаструктуры
- **ОРБИТАЛПАНК / СТАНЦИОННЫЙ** — жизнь на станциях, в невесомости
- **МЕГАСТРУКТУРЫ** — Дайсон-сферы, кольца, орбитальные ободы
- **КОСМИЧЕСКИЙ ФРОНТИР / SPACE-WESTERN** — старатели, фронтир, салунные руины
- **ГЕНЕРАЦИОННЫЙ КОВЧЕГ / АСТРОИНЖЕНЕРИЯ** — поколенческие корабли

### Материя / химия *(нишевая, опционально)*
- **КРИСТАЛЛО-/СИЛИКАТНАЯ «жизнь»**
- **ПРОГРАММИРУЕМАЯ МАТЕРИЯ** — умные материалы
- **ХИМ-ИНДАСТРИАЛ** — реагенты, токсины, кислоты *(пересекается с био/индустрией)*

## Сквозные оси (модификатор поверх фракции/мира)

### Общество / идеология — *как фракция устроена и во что верит*
- **ТЕХНО-ТЕОКРАТИЯ / КУЛЬТ** · **КОРПОРАТОКРАТИЯ** · **ТОТАЛИТАРНАЯ АНТИУТОПИЯ** · **АНАРХО-КОММУНА / МУТУАЛИЗМ** · **ТЕХНО-ФЕОДАЛИЗМ / ДОМА** · **КОЛЛЕКТИВНЫЙ РАЗУМ / УЛЕЙ**

### Сознание / психика — *может быть и фракцией, и слоем*
- **НЕЙРОПАНК** (взлом мозга, BCI) · **СИМУЛЯЦИЯ / ВИРТ** · **ЗАГРУЖЕННЫЕ СОЗНАНИЯ** (цифровое бессмертие) · **ВОСПРИЯТИЕ-НА-ИЗЛОМЕ** (ненадёжная реальность)

### Среда / биом — *фон, задаёт «где происходит»*
- **ЗАТОПЛЕННЫЙ / АБИССАЛЬ** · **ЛЕДНИКОВЫЙ / ДОЛГАЯ ЗИМА** · **ПУСТЫНЯ / ДЮННЫЙ** · **ЗАРОСШИЙ / ДЖУНГЛИ-РЕКЛЕЙМ** · **ПОДЗЕМНЫЙ / БУНКЕРНЫЙ**

### Тон / хоррор — *настроение поверх тех-эстетики*
- **БОДИ-ХОРРОР** (усиливает био/мутации) · **КОСМИЧЕСКИЙ ХОРРОР** (усиливает аномалии) · **ВИРУСНЫЙ / ПАНДЕМИЧЕСКИЙ** (усиливает выживание) · **UNCANNY / ЖУТКИЙ ИИ** (усиливает ИИ/робопанк)

## Как пользоваться

1. Выбрать **среду** и **тон** на весь мир (общий фон).
2. На каждую из 7 фракций назначить **одну тех-эстетику** (можно с парным под-жанром из той же семьи).
3. По желанию навесить на фракцию **идеологию** и/или **ось сознания** для глубины.
4. Проверить, что эстетики контрастны между собой, но объяснимы одним общим местом и одним общим призом мира.
