# Фракции — обзор

Лор происхождения фракций — [[znoy|Сеттинг: Зной]]. Все 7 фракций выросли после одной катастрофы — **Зноя** — каждая вокруг своего обломка старого мира. Общее место мира — великая пустыня; общий приз — уцелевший в её центре схрон **«Колыбель»**.

Каждая фракция держится на **одной главной механике**. Жанр — одна тех-эстетика из [[theme-palette|палитры]] поверх общего фона **ПОСТАПОКАЛИПСИС / ПУСТЫНЯ**.

## Ограничение дизайна

**Запрещено:** механики, создающие новых существ на поле боя из ниоткуда. Все существа играются с руки (у каждого существа на поле есть физическая карта в RFID-ячейке).

## Главные механики — кратко

- **Свора X** (Шакалы): это существо получает `+1` к атаке за каждое **другое** дружественное существо на поле, максимум `+X`. Больше своих — больнее бьёт.
- **Перегрев** (Пепел): в начале вашего хода существо получает `+1` к атаке (видно на правой полосе); **Сброс** наносит цели урон, равный текущей атаке, и возвращает атаку к базовой.
- **Адаптация** (Химеры): впервые пережив урон, существо получает одно постоянное усиление из набора на карте (выбирает игрок). Один раз за игру.
- **Броня X** (Бастион): поглощает `X` урона, прежде чем страдает здоровье; тратится по мере попаданий.
- **Взлом** (Сеть): выключить вражеское существо до вашего следующего хода — оно не атакует, не использует способности и теряет ключевые слова.
- **Цветение X** (Оазис): в начале вашего хода существо восстанавливает `X` здоровья **всем** дружественным существам.
- **Эхо** (Мираж): способность этого существа срабатывает **дважды**.

> Полные правила механик, примеры существ, флейвор и силы героев — в файлах фракций. Принципы баланса под каждую механику — в `CLAUDE.md`.
